# API Testing & Validation Using Jest and Express Validator

A practice project created to learn how to validate API requests and test backend APIs using Jest and Express Validator.

This repository focuses on building reliable Express.js APIs by validating incoming user data and writing automated test cases.

## 🚀 Features

- REST API development using Express.js
- Request body validation
- Input validation using Express Validator
- Automated API testing using Jest
- Testing valid and invalid API requests
- Proper error handling and validation responses
- Organized project structure

## 🛠️ Technologies Used

- Node.js
- Express.js
- Jest
- Express Validator
- JavaScript
- npm

⚙️ Installation
Clone the repository:

git clone https://github.com/Anand-1081/api-testing-validation-using-jest-express-validator.git
Navigate to the project directory:

cd api-testing-validation-using-jest-express-validator
Install the dependencies:

npm install
▶️ Run the Server
